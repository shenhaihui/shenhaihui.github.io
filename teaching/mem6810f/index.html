Course Description

    The class for full-time master students focuses more on the theory and analysis. It covers classical concepts and theories in discrete-event system simulation, mainly including queueing models, random variate generation, input modeling, and output analysis. Moreover, some algorithms for simulation optimization together with their analysis are touched. It also provides basic training on conducting simulation in Microsoft Excel and FlexSim, with applications in logistics and supply chain.
